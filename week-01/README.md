# Week 01 – HTML Document Structure

##  Objective
Reinforce HTML fundamentals by creating a structured, multi‑page website that incorporates:
- Proper document structure (`<!DOCTYPE html>`, `<html>`, `<head>`, `<body>`)
- Navigation between pages
- Text content, lists, and multimedia elements

---

## What Was Done
- Created **four web pages**:
  - `index.html` (Home)
  - `about.html` (About)
  - `contact.html` (Contact)
  - `external.html` (External Links)
- Implemented a **navigation bar** with internal links between all pages.
- Added **external links** to relevant websites (MDN, W3Schools, Unsplash).
- Inserted **two images**:
  - One local image stored in the project folder.
  - One online image sourced from [Unsplash](https://unsplash.com).
- Organized content using **lists** (unordered and ordered).
- Added headings, paragraphs, and comments for clarity.
